# Cute Survivors!

A **Vampire Survivors-style** 3D arena survival game built entirely with Three.js and procedural geometry. No external assets needed — everything is cute, bouncy, and made from primitives!

## Play

Open `index.html` in a modern browser, or visit the GitHub Pages deployment.

## Features

- **Chibi player character** with crown accessory, big eyes, blush, and bouncy walk
- **Cute blob enemies** (slimes) in 4 color variants that swarm toward you
- **Auto-attack AOE pulse** damages all enemies in range
- **XP gems** with glow rings — collect them to level up
- **Upgrade system** — choose 1 of 3 upgrades each level (damage, speed, HP, range, attack speed)
- **Difficulty scaling** — more enemies, faster spawns, and tougher creatures over time
- **HUD** with HP bar, XP bar, kill count, survival timer, and level display
- **Start screen** with high score display
- **Game over screen** with stats and new high score detection
- **Pause** with ESC or P
- **localStorage** high score persistence
- **Visual polish**: camera shake, damage numbers, enemy HP bars, death particles, ambient floating sparkles, squish animations, level-up celebrations

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Move |
| Auto | Attack (AOE pulse) |
| ESC / P | Pause / Resume |
| Mouse Click | Select upgrade on level-up |

## Tech Stack

- **Three.js** (v0.160 via CDN)
- Single HTML file, no build step
- Google Fonts (Nunito)
- All assets procedurally generated

## Aesthetic

Pastel color palette with chibi-style characters, soft rounded shapes, floating hearts and stars, and a cute meadow arena. Everything is designed to be adorable!
